1.	Berdasarkan prinsip SOLID, jelaskan apa pengertian dan keuntungan dari penerapan prinsip berikut pada pengembangan perangkat lunak berikut
    a)	Prinsip single responsibility
        Prinsip Single responsibility adalah prinsip pengembangan perangkat lunak yang memastikan bahwa setiap class hanya memiliki satu tugas utama saja.

        Keuntungan
        Keuntungannya adalah code menjadi lebih rapi, performance menjadi lebih cepat, jumlah line pada method lebih sedikit sehinggan membuat code lebih enak dimaintain dan penggunaan resource memory lebih sedikit.
    
    b)	Prinsip open/closed
        Prinsip open/closed merupakan prinsip yang digunakan pada sebuah source code dimana kita bisa merubah dengan cara menambah entitas tanpa merubah source code yang sudah di tentukan sebelumnya. Sehingga tidak perlu lagi mengubah sumber source code yang sifatnya tertutup.

        Keuntungan
        Keuntungan nya adalah ketika ada persyaratan yang ingin diubah, cukup menambahkan behavior dari suatu modul dengan menambah kode baru, bukan dari mengubah kode lama yang telah bekerja. Ini menjadikan OCP meringankan maintenance kode dan meminimalkan resiko dari merusak kode yang telah ada.
